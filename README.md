# The-Telecommunication-analysis

This project presents an interactive Power BI dashboard analyzingthe performance of major telecommunication operators in Nigeria from 2020 to 2025. The analysis coveres financial performance, subscriber growth, data usuage, and voice traffic across key regions to uncover trends and business insights.

Business Problem

Tele communication operators need clear visiblility into regional and operator-level performance to optimize revenue, control cost, and improve customer acquisition. 

This project answers:

1.Which region and operators generate the most revenue and profit?
2. How do expenses, subscribers, data usage, and voice traffic impact performance?
3.What strategies can drive sustainable growth?

Data Overview

Time period: 2020-2025

Regions: Abuja, Enugu, Kano, Lagos, Port Harcourt.

Metrics: Revenue, Profit, Expenses, Subscribers, Data usage, and Voice traffic.

Dashboard Features

1. Revenue, profit, and expenses breakdown by region and operator
   
2. Year-over-year trend analysis
   
3. Operator performance comparison
   
4. Correlation insight between data usage, subscriptions,and revenue
   
5. Intractive filters for region, operator and year


Key Insights

1. Total revenue: #21.24 trillion | Total profit: #7.45 trillion

2. Strong correlation between revenue and expenses, and revenue and subscribers

3. High data usage and voice traffuc  generally lead to higher revenue

4. Some regions show high revenue despite low data usage, indicating pricing or mixed effects

Tools & Skills Demonstrated

1. Power BI
   
2. Dax calculations
   
5. Trends and correlation analysis

6. Business intelligence & storytelling

Outcome

The dashboard provides actionable insights that can help telecom operators improve network investment decisions, pricing strategies, and regional growth plans.
